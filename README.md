# 🎬 Movie Ticket Booking System

A terminal-based Movie Ticket Booking System built in Python.  
Micro-Project | Alliance University — Alliance College of Engineering and Design

---

## 📌 Features

- 🎥 View all available movies with showtimes and seat availability
- 🎟️ Book tickets with real-time seat tracking
- ❌ Cancel bookings using Booking ID
- 📋 View all your bookings
- 💾 Persistent storage using JSON (bookings saved between runs)
- ✅ Input validation and error handling throughout

---

## 🛠️ Tech Stack

- **Language:** Python 3.10+
- **Libraries:** `json`, `os`, `random`, `datetime` (all built-in — no installation needed)

---

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/movie-ticket-booking.git
   cd movie-ticket-booking
   ```

2. **Run the program**
   ```bash
   python main.py
   ```

That's it! No external libraries needed.
